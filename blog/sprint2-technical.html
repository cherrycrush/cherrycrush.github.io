Best practice: "Class" and "id"

When writing HTML it is important to follow responsive web design practises. This topic will illustrate the best practices of the class and id attributes - when and how to use them.

Classes are a method for selecting multiple HTML elements and applying style properties across them all. Javascript and CSS can use these classes to manipulate more HTML elements at once. This offers you more flexibility, and prevents the hassle of having to define each element and style them individually. The id attribute is a method for selecting individual HTML elements. This is when you want to manipulate one specific element.
